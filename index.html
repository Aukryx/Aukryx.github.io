Lem-in Ant Colony Pathfinder

Algorithm (Go)

Lem-in is a graph-processing project built in Go that efficiently routes ants through a network of tunnels. The algorithm finds the optimal distribution of ants across multiple paths to minimize the total number of turns while respecting constraints: only one ant per room and each ant moves once per turn.

Go Graph Theory Pathfinding Optimization
